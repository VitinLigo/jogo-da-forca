# 🎮 Jogo da Forca — Linguagens de Programação
Um jogo da forca no terminal desenvolvido em **Java**, onde o jogador tenta adivinhar linguagens de programação famosas antes de perder todas as chances!
---
## 🕹️ Como funciona
- O jogo escolhe uma linguagem de programação aleatória
- O jogador tem **5 tentativas** para adivinhar a palavra correta
- A cada erro, o número de tentativas restantes é exibido
- Ao acertar ou perder, o jogador pode escolher jogar novamente
---
## 🚀 Funcionalidades
- ✔️ Seleção aleatória de linguagem de programação
- ✔️ Controle de tentativas com feedback a cada erro
- ✔️ Opção de jogar novamente ao fim de cada rodada
- ✔️ Entrada do usuário sem distinção de maiúsculas/minúsculas

## 🖥️ Exemplo de execução
```
Olá, seja bem vindo ao jogo da forca de linguagens de programação! A linguagem tem 4 letras.
ruby
Você errou, tem apenas mais 4 tentativas.
javascript
Você errou, tem apenas mais 3 tentativas.
java
Parabéns! Você acertou. A linguagem era java!
Você deseja jogar novamente? (S/N)
```

## ⚙️ Como executar
1. Clone o repositório:

2. Abra o projeto em uma IDE:
   - IntelliJ IDEA
   - Eclipse
   - VS Code (com extensão Java)
3. Execute o arquivo:
```

```
